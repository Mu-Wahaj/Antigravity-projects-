# Portfolio

A modern developer portfolio showcasing projects, skills, and contact information.

## Features

- Interactive portfolio sections
- Responsive design
- Smooth scrolling animations
- Project showcase

## Tech Stack

- React
- Vite
- CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/` - application source code
- `public/` - static assets
- `src/components/` - reusable UI components
