# Brickell Dental Clinic

A modern, premium dental clinic website for a cosmetic and general dentistry practice in Miami. 
Designed with a clean healthcare aesthetic, it features a professional and visually appealing interface that emphasizes patient comfort and high-end services.

## Technologies Used
- React
- Vite
- Vanilla CSS
- React Router DOM
- Framer Motion
- React Icons

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
