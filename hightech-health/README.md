📝 About
A clean, professional website for an aesthetic medicine clinic built with React and Vite. Features elegant UI, smooth animations, and full responsiveness.

✨ Key Features
🏠 Home Page - Hero section, services overview, testimonials

💆 Services - Treatment cards with details & pricing

👩‍⚕️ About - Doctor profiles & clinic philosophy

📅 Booking - Appointment request form

📞 Contact - Location, contact form, WhatsApp integration

🎨 Premium UI - Soft gradients, smooth animations, gallery

🛠️ Built With
React 18 - UI framework

Vite - Build tool

Framer Motion - Animations

Lucide React - Icons

CSS3 - Styling (no Tailwind)

🚀 Quick Start
bash
# Clone repo
git clone https://github.com/Mu-Wahaj/aesthetic-clinic.git

# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
