# Real Estate Website

A Next.js real estate application for browsing listings, filtering properties, and contacting agents.

## Features

- Property listings and detail pages
- Contact form
- Server-side API routes
- Responsive layout

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## Project Structure

- `app/` - Next.js application routes and pages
- `components/` - reusable UI components
- `lib/` - helper functions and API utilities
- `prisma/` - database schema
